#  Weather App

Aplicação simples em **Node.js + TypeScript** para consultar informações climáticas em tempo real através de uma API.

---

##  Tecnologias

* Node.js
* TypeScript
* API de clima
* ts-node

---

##  Configuração (.env)

Crie um arquivo `.env` na raiz do projeto com:

```
API_KEY=052a15caea69d275dadb64bfc6b71130
PORT=3000
```

---

##  Como Executar

```
npm install
npx ts-node src/server.ts
```

A aplicação estará disponível em:
http://localhost:3000

---

##  Funcionalidades

* Buscar clima por cidade
* Retornar temperatura, clima, umidade e vento
